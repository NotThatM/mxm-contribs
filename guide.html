Musixmatch Contributor Viewer — User Guide

Step 1: Install the Script

  1. Go to the install website: bryyce19.github.io/mxm-contribs
  2. Click the Install Script button.
  3. Once installed, you’re ready to use the script on Musixmatch.

Step 2: Using the Script

Where to Start

Navigate to any mission or non-mission lyrics page on musixmatch.com.

Accessing Contributor Info

  1. Click the icon on the lyrics card.
  2. The first time, a new orange button will appear.
  3. Click the button to load contributors.

Features & Filters

Filter by Edit Type

Filter contributors by edit type (currently lyrics and tagging).

Recent Contributor Highlight

The orange-colored name indicates the most recent contributor.

Contributor Roles

IconRole
Specialist
Curator
Graduate/Rookie

User Interface Controls

  • Theme Toggle: Top right corner, switch between dark/light mode.
  • Close Viewer: Click the X icon at the top right.
  • Manual Scroll: Use if scroll bar fails (known bug).

Warnings & Errors

The viewer only works on valid tool/mission pages. If the script detects an unsupported page, an error will appear.
